# Presency App

React Native app for managing Presency clients and leads.

## Run on your phone in 2 minutes

```
git clone https://github.com/kiing76bb-code/presency-native-app.git
cd presency-native-app
npm install --legacy-peer-deps
npx expo start
```

Then scan the QR code with Expo Go on your phone.

## Stack
- Expo SDK 51 + Expo Router
- Supabase (presency-platform project)
- React Native SVG icons
- Dark luxury gold aesthetic

## Screens
- Home: MRR, stats, quick actions
- Clients: full list with tier badges
- Client Detail: stats, services, contact
- Leads: form submissions with call/text buttons
- Settings: platform links + Stripe payment links
